Recursive Descent Parser for the token_scanner. 

How to compile source code and create executable:
1.	download and install MinGW
2.	open command prompt
3.	set path using this command: C:\> set PATH=C:\MinGW\bin;%PATH%
4.	change directory to token_parser folder
5.	compile using this function and create executable file: g++ main.cpp -o parser.exe
6.	program should be compiled and parser.exe should be created in token_parser folder
7. 	to run the program/executable(assuming you are still in same directory) type in command prompt: parser.exe (file name). e.g: parser.exe input.txt
